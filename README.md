# Smash Trophy Maker
**Smash Trophy Maker** is a website that lets you create your own **Super Smash Bros trophies**. 
Upload an image of your favorite character, customize its appearance, and make it look like a real in-game trophy!

#  How to Use

1. Go to the [Smash Trophy Maker website](https://juanp-0.github.io/Smash-Trophy-Maker/).  
2. Upload an image of your favorite character.  
3. Customize your trophy using the Trophy Dashboard options.  
4. Enjoy your personalized Super Smash Bros trophy!
